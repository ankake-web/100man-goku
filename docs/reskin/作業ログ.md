## 0. プロジェクト概要

- カタン風・戦国テーマのボードゲーム「**百万石**」（旧「100万石」）を、ChatGPT 生成の
  **フィギュア／ミニチュア模型風アート**に総差し替えするブランチ。
- 技術: TypeScript + Vite + バニラ DOM/SVG。サイコロのみ three.js（WebGL, `diceGL.ts`）。
- 起動: `npm run dev`（既定 5173、使用中なら 5174 等にフォールバック）。

## 1. アセットの仕組み（差し替え方法・最重要）

画像参照は**中央マニフェスト経由**で、固定ファイル名を import している。
**差し替え＝ `src/assets/` の同名ファイルを上書き**するだけ。欠損時は `placeholder()` で
壊れ画像を出さずフォールバックする（クラッシュしない）。

- ゲーム用マニフェスト: [`src/assets/manifest.ts`](../../src/assets/manifest.ts)
  - `ASSETS.*` が唯一の画像参照元。`assetImg()` / `placeholder()` / `setIconLabel()` を提供。
- 生成ツール定義: [`tools/assetgen/manifest.mjs`](../../tools/assetgen/manifest.mjs)（寸法・英語プロンプト・スタイル接頭辞）
- 仕様の正典: [`ART_ASSET_SPEC.md`](ART_ASSET_SPEC.md) / 画風と発注文: [`画像生成リスト.txt`](画像生成リスト.txt)

### ChatGPT 画像の取り込みフロー
1. チャッピー（ChatGPT）に画像を作ってもらい、リポジトリ直下 `image/`（gitignore）へ入れる。
2. 取り込みツールで正式名へ配置（中身で確定したマッピング）:
   ```
   cd tools/assetgen
   node place-chatgpt.mjs            # 変換して chatgpt-staged/ にプレビュー生成
   node place-chatgpt.mjs --commit   # OK なら src/assets/ へ本適用
   ```
   - マッピングは `place-chatgpt.mjs` の `ASSIGN`（index = mtime 昇順の通し番号）にハードコード。
   - **白背景の透明化はフラッドフィル方式**（縁につながった近白だけ抜き、中央の白＝旗/紙/白壁は残す）。
     `isBgWhite`: `min(r,g,b) >= 226 && (max-min) <= 18`（低彩度ガードで色付き被写体を保護）。
   - **グレーの色違いベース**（砦/城/船/天守）は `sharp.tint` で朱/藍/紫/山吹へ自動着色。
   - 背景 bg-*（横長 JPG）は透明化せず flatten のみ。
3. git 書き込みはオブジェクトストアが sandbox 外のため、コマンドに `dangerouslyDisableSandbox: true` が必要。
