# 100万石 リスキン用語集（カタン → 戦国テーマ）

このファイルは「100万石」リスキンの**唯一の正典**。表示文字列（ユーザーに見える日本語ラベル）だけを置換する。
内部識別子・enum・キー・変数名・関数名・アセットのファイル名・テストのロジックは**一切変えない**。

## 大原則（厳守）
1. 変えるのは「ユーザーに表示される文字列(ラベル)」のみ。`ResourceType.WOOD` の `'wood'` キーや `'forest'` 等の内部値は不変。
2. 複合語・文脈依存語を**先に**処理し、その後に単独語を処理する（誤置換防止）。
3. 数値・ダイス・確率・絵文字（⚔🛤🏛🌾🛡🐎🧱⚓🌲🐑⛏📈🎲★ 等）は変えない。絵文字に続く日本語テキストだけ置換する。
4. ロジックで比較される文字列は変えない。特に `main.ts` の `v === '指定'`（fixed/random 判定）は**変更禁止**。
5. アセットのファイル名（`knight-basic.png` 等）・import パス・`ASSETS.*` のキーは不変。`alt` テキスト等の表示文字列だけ置換可。
6. コメント内の用語は、純粋な用語呼称であれば一貫性のため更新してよい（任意・低優先）。ただし構造説明・内部キー言及・分かりやすさを損なう箇所は無理に変えない。コードの意味を壊さないことが最優先。
