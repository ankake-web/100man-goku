# BGM 候補・利用規約メモ

カタン風オンライン対戦のBGMに関するメモ。現状の実装と、将来「実音源」を同梱する場合の候補・規約確認事項をまとめる。

## 現状の実装（2026-06 更新）

BGMは **実音源(`public/bgm`)を優先再生し、未配置／読込失敗時は手続き生成（WebAudio オシレータ）の
オリジナル旋律へフォールバック**する二段構え。実音源が無くても必ず何か鳴り、ゲームは壊れない。

- 実装場所: `src/main.ts` の `BGM_TRACKS`（`bgmStart` → 実音源 `HTMLAudioElement` / 失敗時 `bgmStartProcedural`）
- 3種類（名前 / 期待ファイル名 / 採用候補）:
  1. **開拓の朝** — `/bgm/village_fete.mp3` — PeriTune「Village_Fete」
  2. **港町の酒場** — `/bgm/portside_cafe.mp3` — PeriTune「Portside Café」
  3. **静かな夜** — `/bgm/nocturnal_bloom.mp3` — PeriTune「Nocturnal_Bloom」
- 選択は「ゲームメニュー → BGM 曲」。選択値は `localStorage['catan_bgm_track']` に保存し次回も同じ曲。
- 音OFF（BGM OFF）時は鳴らさない。曲切替はフェードアウト→フェードインで自然に差し替え（実音源・手続き生成の両方）。
- 実音源の配置手順は `public/bgm/README.md` を参照。**音源ファイル自体は未同梱**（理由は下記「今回の結論」）。

### PeriTune 全体の傾向（※公式の最新規約で要再確認）

- 多くの楽曲が **CC BY 4.0**（**クレジット表記を条件に**商用含め利用可・加工可）。表記例: `Music: PeriTune`（作者名・配布元URL）。
- ただし**曲・配布プラットフォームによって条件が異なる**ことがある（YouTube Audio Library 版や他サイト再配布版など）。各曲ページの記載を最優先。
- 楽曲ファイル単体の再配布・販売は不可。YouTube 等への Content ID 登録は不可。

- 中世・酒場・ケルト: 「Celtic」「Medieval」「Tavern」系
- 夜・静かなファンタジー: 「Fantasy」「Ambient」「Night」系のスローテンポ曲

## 音源を同梱する際の注意

- ループ再生に向く（イントロ/アウトロの無い、または自然に繋がる）素材を選ぶ。
- ファイルは `public/bgm/` 等へ置き、`import` ではなく URL 参照でロードする（バンドル肥大化回避）。
- 規約で「クレジット表記」が必要な場合、ゲーム内（設定画面やクレジット）に作者名・配布元を明記する。
- 規約で「素材そのものの再配布」が禁止されている場合、リポジトリ公開時の扱い（同梱可否）を確認する。
- 音量はゲーム進行の邪魔にならないよう控えめ（現状の既定 `_bgmVolume = 0.07` 相当）に正規化する。
- 採用したら本ファイルの「候補曲」表に曲名・作者・配布元・ライセンス・**ライセンス確認日**を必ず記録する。

## 今回の結論

- **実音源は未同梱**。理由: 作業環境がオフラインで、PeriTune の各曲ページ・利用規約・音源ファイルへアクセス
  できず、ライセンス確認もダウンロードもできなかったため（規約未確認のまま勝手に同梱はしない方針）。
- 代わりに、**実音源を `public/bgm/` に置けばそのまま実ファイル再生に切り替わる**コード（`HTMLAudioElement`
  ＋手続き生成フォールバック）と、配置手順（`public/bgm/README.md`）、本規約メモを整備した。
- 次の手順: 人が PeriTune の3曲ページと規約を確認 → 問題なければ規定のファイル名で `public/bgm/` に保存 →
  本ファイルの「採用候補曲」表に**確認日・クレジット要否**を記録 → 必要ならゲーム内にクレジットを表示。
- 候補曲の条件が合わない場合は同梱せず、上記「代替」から雰囲気の近い曲を選び直す。
