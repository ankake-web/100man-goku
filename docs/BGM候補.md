# BGM 候補・利用規約メモ

カタン風オンライン対戦のBGMに関するメモ。現状の実装と、将来「実音源」を同梱する場合の候補・規約確認事項をまとめる。

## 現状の実装（2026-06）

著作権・利用規約の確認が取れる実音源を今回は同梱していないため、**手続き生成（WebAudio オシレータ）によるオリジナル旋律**を3種類用意した。手続き生成なので第三者の著作物を含まず、ライセンス上の制約がない。

- 実装場所: `src/main.ts` の `BGM_TRACKS`（`bgmStart` / `setBgmTrack`）
- 3種類:
  1. **開拓の朝** — 明るくほのぼの（Cメジャー・ペンタトニック寄り）
  2. **港町の酒場** — 中世・ケルト風の軽快なリール（Aドリアン）
  3. **静かな夜** — 落ち着いたファンタジー（Aマイナー、ゆっくり）
- 選択は「ゲームメニュー → BGM 曲」から。選択値は `localStorage['catan_bgm_track']` に保存し次回も同じ曲。
- 音OFF（BGM OFF）時は鳴らさない。曲切替時はフェードアウト→フェードインで自然に差し替え。
- AudioContext が使えない／読み込めない環境でもゲームは壊れない（try/catch でフォールバック）。

実音源（mp3/ogg 等）に差し替える場合は、`BGM_TRACKS` を `HTMLAudioElement`／`AudioBufferSourceNode` ベースの管理へ置き換えれば、選択UI・保存・フェード・音OFF連動の構造はそのまま使える。

## 音源を同梱する際の注意

- ループ再生に向く（イントロ/アウトロの無い、または自然に繋がる）素材を選ぶ。
- ファイルは `public/bgm/` 等へ置き、`import` ではなく URL 参照でロードする（バンドル肥大化回避）。
- 規約で「クレジット表記」が必要な場合、ゲーム内（設定画面やクレジット）に作者名・配布元を明記する。
- 規約で「素材そのものの再配布」が禁止されている場合、リポジトリ公開時の扱い（同梱可否）を確認する。
- 音量はゲーム進行の邪魔にならないよう控えめ（現状の既定 `_bgmVolume = 0.07` 相当）に正規化する。
- 採用したら本ファイルの「候補曲」表に曲名・作者・配布元・ライセンス・**ライセンス確認日**を必ず記録する。

## 今回の結論

- 実音源の規約確認・同梱は本タスクのスコープ外（オフライン環境で規約確認ができないため）。
- まず **3種類のBGM選択UI＋差し替え可能な audio 管理構造**を手続き生成BGMで実装済み。
- 実音源を入れる場合は、本メモの候補・注意に沿って規約を確認し、`BGM_TRACKS` の生成部を音源ロードへ差し替える。
