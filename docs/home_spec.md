# ホーム画面仕様 — カタン

## 概要

ゲーム開始前に表示される設定画面。モード選択・プレイヤー設定を行い「ゲーム開始」でゲームに遷移する。

## コンポーネント詳細

### タブ切り替え

| タブ | 説明 |
|------|------|
| CPU 対戦 | ローカルで人間 vs CPU |
| オンライン対戦 | WebSocket ルームマッチング（将来実装） |

デフォルトは「CPU 対戦」を選択状態とする。

---

### CPU 対戦フォーム

#### プレイヤー名

- テキスト入力（`<input type="text">`）
- デフォルト値: `"プレイヤー1"`
- 最大 20 文字
- 空欄の場合はデフォルト値で補完

#### CPU 数

- ラジオボタン: 1 / 2 / 3
- デフォルト: 1
- プレイヤー数 = CPU 数 + 1（人間 1 名固定）
- 最大 4 人プレイ（CPU 3 + 人間 1）

#### CPU の強さ

- ラジオボタン: 弱 / 普通 / 強
- デフォルト: 普通
- 全 CPU に同一の強さを適用

---

### オンライン対戦フォーム（スタブ）

- ルーム作成ボタン / ルーム参加ボタンを表示
- クリック時は「近日公開予定」のメッセージを表示
- 実装は将来フェーズとする

## データモデル

```typescript
interface HomeConfig {
  mode: 'cpu' | 'online';
  playerName: string;        // 人間プレイヤー名
  cpuCount: 1 | 2 | 3;      // CPU 数
  cpuDifficulty: AiDifficulty; // 'weak' | 'normal' | 'strong'
}
```

---

## 遷移フロー

```
アプリ起動
    │
    ▼
ホーム画面表示
    │
    │ [ゲーム開始] クリック
    ▼
GameState 初期化（HomeConfig に基づく）
    │
    ▼
ゲーム画面表示（ホーム画面を非表示）
```

---

## AI 強さの定義

| レベル | 説明 | 行動方針 |
|--------|------|----------|
| 弱 | 初心者向け。ランダム寄りの選択 | セットアップ: ランダム頂点選択。TRADE_BUILD: ランダムアクション。ROBBER: ランダムタイル |
| 普通 | バランス型。現行ロジック | 生産スコア優先の建設。バンク交易の活用 |
| 強 | 上級者向け。戦略的判断 | 勝利点効率を最大化する優先度付き戦略。即勝利チェック。強盗の最適化 |
