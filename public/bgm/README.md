# BGM 音源の配置

このフォルダ（`public/bgm/`）に以下の3ファイルを置くと、ゲームのBGMが手続き生成から**実音源**に切り替わります。
ファイルが無い／読み込めない場合は、自動で**手続き生成BGM（フォールバック）**が鳴るため、ゲームは壊れません。

| トラック | 期待ファイル名 | 採用候補曲 |
|---|---|---|
| 開拓の朝 | `village_fete.mp3` | PeriTune「Village_Fete」 |
| 港町の酒場 | `portside_cafe.mp3` | PeriTune「Portside Café」 |
| 静かな夜 | `nocturnal_bloom.mp3` | PeriTune「Nocturnal_Bloom」 |

## 配置手順

1. 各曲ページと **PeriTune の利用規約**を確認する（ゲーム同梱可否・Web利用・商用利用・クレジット要否・加工可否・楽曲ファイル単体再配布の禁止 等）。
2. 規約上問題なければ、上記のファイル名で `public/bgm/` に保存する（`vite` が `/bgm/<name>.mp3` で配信）。
3. ファイル名は `src/main.ts` の `BGM_TRACKS[].file`（`/bgm/...`）と一致させる。
4. 確認結果（曲名・作者・配布元・ライセンス・**確認日**）を `docs/BGM候補.md` に記録する。
5. 規約でクレジット表記が必要な場合、ゲーム内（メニュー等）または README にクレジットを残す。

## 注意

- ループ再生されるため、ループに向く（自然に繋がる）音源を推奨。
- 音量はコード側で控えめに調整済み（`bgmAudioVol()`）。
- **本リポジトリには音源ファイルを未同梱**（オフライン環境で規約確認・取得ができなかったため）。
- 楽曲ファイル単体の再配布・販売、YouTube 等への Content ID 登録は禁止。
- 将来有料公開する場合は、音源ファイルが直接抽出されにくい形（難読化・サーバ配信等）の検討が必要。
